How Metrics Are Calculated (6 Metrics)

1) Feature Vector Per Movie

Each movie is converted to one vector using: genre binary flags + normalized rating + normalized release year.

movie_vector = [genre_1..genre_n, rating/5, (year-minYear)/(maxYear-minYear)]

2) User Preference Vector

Your selected genres are 1, others 0. Then rating/year targets are set from selected option.

recent_release → ratingTarget=0.70, yearTarget=1.00 | high_rating → ratingTarget=1.00, yearTarget=0.60

3) Preference Weights

Weights are applied before KNN so preference changes the actual distance calculation.

recent_release → ratingWeight=0.8, yearWeight=2.6 | high_rating → ratingWeight=2.0, yearWeight=0.8

4) KNN Distance + Top 10

Euclidean distance is computed to all movies, sorted ascending, and nearest 10 are returned.

distance = sqrt(sum((weighted_movie_i - weighted_user_i)^2))

5) Chart Summary Metrics

Best Match = minimum distance, Average Distance = mean of top 10, Distance Range = max-min.

best=min(d), avg=sum(d)/10, range=max(d)-min(d)

6) Match Quality (Stars + Remark)

Quality labels come from distance thresholds used in recommendation table.

d < 0.3 → ★★★★★ Perfect | 0.3 ≤ d < 0.5 → ★★★★ Excellent | 0.5 ≤ d < 0.7 → ★★★ Good | d ≥ 0.7 → ★★ Fair
